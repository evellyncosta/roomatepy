# roomatepy
Este projeto representa uma Prova de Conceito (PoC) com o principal objetivo de avaliar e testar as funcionalidades e a eficiÃªncia do CacheTools, uma biblioteca Python para gerenciamento de cache.

# Tecnologias
#### Python
#### MariaDB
#### Docker e Docker-Compose
#### FastAPI
#### Cachetools
#### Poetry
#### Alembic


# Setup
Este projeto usa docker e docker-compose, para rodar o projeto, execute o comando

`docker-compose build`

e em seguida

`docker-compose up`
